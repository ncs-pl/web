Why nWT?

nc0's Web Toolkit (shortened as nWT) is a set of tools, guides, and libraries, to make it simpler and quicker to program and design applications and documents on the World Wide Web.

Using only accepted Web standards, while keeping support for older Web browsers, nWT aims for performance and intuitive API. nWT is available under a free license contract. Sources Mailing list nc0
