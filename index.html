nc0's Web Toolkit (nWT) is a toolbox for developers, designers, and others to quickly prototype and iterate on hypertext documents on the World Wide Web, in a responsive, interactive, and elegant manner.

Fluid Layout

A functional CSS library providing an elegant and responsive layout system (fluid layouts) that automatically adapts its layout given the available space and the size occupied by its inner content or children.
